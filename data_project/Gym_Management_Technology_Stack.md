# فناوری‌های مورد استفاده در پروژه مدیریت باشگاه بدنسازی

## مقدمه

این پروژه به صورت Full Stack توسعه داده می‌شود؛ به این معنا که رابط
کاربری و منطق سمت سرور در یک پروژه واحد پیاده‌سازی خواهند شد. هدف از این
انتخاب، کاهش پیچیدگی، افزایش سرعت توسعه و سهولت نگهداری پروژه است.

## تکنولوژی‌های اصلی

### Next.js 15

فریمورک اصلی پروژه که برای پیاده‌سازی رابط کاربری و بخش Backend (Route
Handlers و Server Actions) استفاده می‌شود.

**دلایل انتخاب** - یکپارچگی Frontend و Backend - ساختار مدرن App
Router - پشتیبانی از Server Components - مناسب برای توسعه سریع پروژه

### TypeScript

برای افزایش خوانایی، جلوگیری از خطاهای رایج و توسعه‌پذیری بهتر پروژه
استفاده می‌شود.

### PostgreSQL

پایگاه داده اصلی پروژه.

**دلایل انتخاب** - رایگان و متن‌باز - عملکرد مناسب - سازگار با Prisma -
مناسب برای پروژه‌های کوچک و بزرگ

### Prisma ORM

لایه ارتباطی بین برنامه و پایگاه داده.

**مزایا** - مدیریت Migration - Type Safety - توسعه سریع Queryها -
نگهداری آسان

### Tailwind CSS

برای طراحی رابط کاربری.

**مزایا** - سرعت بالا - کدنویسی منظم - طراحی واکنش‌گرا (Responsive)

### React Icons

کتابخانه آیکون‌های رابط کاربری.

**مزایا** - مجموعه‌ای بزرگ از آیکون‌ها از منابع مختلف (Feather، Font
Awesome، Material Design و...) - وزن پایین با Tree Shaking - سادگی
استفاده به صورت کامپوننت React

### React Hook Form

برای مدیریت فرم‌ها.

### Zod

برای اعتبارسنجی اطلاعات ورودی کاربران.

### Server Actions

برای انجام عملیات سمت سرور مانند: - ثبت ورزشکار - ثبت پرداخت - مدیریت
مربیان - مدیریت تجهیزات - ثبت سفارش مکمل

بدون نیاز به ایجاد API جداگانه برای هر عملیات.

## احراز هویت

برای ورود کاربران از Auth.js (NextAuth v5) با روش Credentials (ایمیل و
رمز عبور) و نشست‌های JWT استفاده خواهد شد. اطلاعات کاربران از طریق Prisma
از پایگاه داده خوانده می‌شود. سیستم دارای سه نقش اصلی است: - مدیر - مربی
- ورزشکار

هر نقش فقط به امکانات مجاز خود دسترسی خواهد داشت.

## ساختار کلی سیستم

``` text
Next.js
├── Frontend (React)
├── Server Actions
├── Route Handlers
└── Prisma
        │
   PostgreSQL
```

در آینده انتخاب شده است و برای پیاده‌سازی پروژه مدیریت باشگاه بدنسازی
